## 프로젝트 소개
TalentShare는 각자 자유롭게 정보를 공유할 수 있는 앱입니다.
자신이 잘 알고 있는 분야, 자신만이 알고 있는 정보, 공유하고 싶은 정보 등를 작성하여 자유롭게 공유할 수 있으며
앱 사용자들은 게시글을 통해 정보를 얻을 수 있습니다.
